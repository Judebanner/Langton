
Langton's Ant Simulation - Fixed Version

✅ What Was Fixed:
- Removed broken JavaFX SDK jars (they were corrupted or empty).
- You need to re-download JavaFX SDK 17 and place it under: lib/javafx-sdk-17/lib/

📦 Where to get JavaFX SDK:
Download from: https://gluonhq.com/products/javafx/

⚙️ How to run (from terminal):
java --module-path lib/javafx-sdk-17/lib --add-modules javafx.controls,javafx.fxml MainApp

💡 Tip:
Use IntelliJ or VSCode with JavaFX support for easiest configuration.
